## 🖥 Overview

The **BI-Dashboard** is a free, ready-to-use module that helps Odoo users visualize their key business data with a beautiful, interactive dashboard.
It is designed to work seamlessly with Odoo Community & Enterprise versions.

## ✨ Key Features

- 📊 Customizable dashboard widgets
- ⚡ Real-time KPI updates
- 🔧 Plug-and-play installation
- 💜 Fully compatible with Odoo 19

## 🚀 Installation

1. Download the ZIP file from https://apps.odoo.com/apps/modules/19.0/synconics_bi_dashboard.
2. Extract it into your Odoo `addons` folder.
3. Install required python packages mentioned in `requirements.txt` with the use of `pip`.
4. Go to **Apps** in Odoo and click **Update Apps List**.
5. Search for “Synconics BI-Dashboard” and install it.
